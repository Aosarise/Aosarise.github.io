依赖

需要配置三个依赖。连接池可以替换成其他的

配置文件

注意点：

  1. 需要配置的有三项：映射文件、实体类别名、数据源
  2. 映射文件可以配置多个，也可以配置成一个包
  3. 实体类别名的用处：写mybatis的mapper.xml文件时候写rezultType的时候不用写前面的包的信息
  4. 数据源配置有：类型、mysql驱动、url username password //这些都是java连接数据库
  5. 因为spring有自动提示，username需要注意。不是data-username。

dao文件

Dao，是Data Access Object，是数据访问对象是一个面向对象的数据库接口
简单来说就是直接操作数据库的类，下一部分可以看具体实现
需要实现两个文件：接口(提供可以使用)，xml(写具体的sql语句)
接口文件需要用: @Mapper注解
接口文件：EmployeeDao
两个文件应该放在同一个包里。

xml文件 就mybatis的那种写法没啥特别的

idea里mybatis配置文件注意事项

一般来说idea只会把resource里的.xml文件打包到对应的地方
springboot是用maven来管理的，在pom.xml里可以配置把源码文件夹里的.xml文件打包到正确的地方。

事务

即让整个方法体成为一个事务，具有原子性。要么全做、要么全不做。
只需要两个注解
@EnableTransactionManagement 放在启动类前，开启事务管理

@Transactional 放在事务之前，也就是方法体前面。实现方法的原子性

然后就可以填一些信息了

然后注意看这个页面，是会报错的，点进去载入jdk就可以了
然后text connection测试连接是否成功。
