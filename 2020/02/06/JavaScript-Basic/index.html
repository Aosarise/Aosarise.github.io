变量与对象

声明 :let是 坠 吼 的 ,var可以重复声明一个变量，最好别用😄。
另外JavaScript是弱类型语言。所以声明完一个变量之后，也是可以随便覆盖的。
常见类型：数值、字符串、boolean、函数、对象。

数组

首先数组是特殊的对象–即对数组使用typeof返回的是object
我觉得数组还挺难的，因为弱类型，所以其实数组什么都能放。
声明：

js有一些数组的API:

添加删除

  • 增加元素(结尾处): push()
  • 删除最后一个元素:pop()
  • 头部的 插入:unshift() 删除:shift()

数组属性

  • 数组长度:length 这是个属性，不是方法
  • 判断对象是否是数组:Array.isArray()

数组中间截取或删除

  • 拼接数组:splice(int,int,...args)
    第一个参数为添加新元素的位置。第二个参数为删除多少元素。其余参数为要添加的新元素。
  • 合并数组:contat()
    它总是返回一个新数组。可以使用任意数量的数组参数。它不会更改现有数组。
  • 裁剪数组:slice()
    第一个参数为起点(包括)，第二个参数为终点(不包括，且可不写)。它不会更改现有数组。

数组数据操作

  • 转换成字符串:join(separator)
    separator:可选 指定分隔符。
  • 小写:toLowerCase() 大写:toUpperCase()
  • 数组倒序:reverse()
  • 数组排序:sort()
    默认为字典序。可自定义 com(a,b) a<b返回负值//即a在b前

有一件很狗的事情，就是可以直接往最后一个地方添加，也可以添加元素。
