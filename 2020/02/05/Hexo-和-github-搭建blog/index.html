    Hexo 和 github 搭建blog

    Sarise‘s Blog
    2020-02-05
    关于建blog
    Github, Hexo

    安装node.js

    1. 按照流程安装就可，记录一下安装的目录。（下载页面http://nodejs.cn/download/）

    2.Macos 配置环境变量(windows 大概不需要可以直接用git工具的样子)

    我的 bash_profile就在user目录下 直接 vim .bach_profile

    添加PATH=$PATH:/usr/local/bin/npm:/usr/local/bin/node

    (:后面为npm node 安装路径, 然后在macos里:就是分隔号)

    (也可以合理地花里胡哨一下 export成变量啥的)

    3.淘宝镜像

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org

    安装git

    本机目录 /usr/bin/git

    设置全局用户名 全局邮箱

    (git config —list 可以查看当前信息

    第三行为大小写敏感设置

    生成密钥

    1. code: 生成密钥后，在本地的/Users/当前电脑用户/.ssh目录下会生成两个文件id_rsa、id_rsa.pub。
      id_rsa文件保存的是私钥，保存于本地，id_rsa.pub文件保存的是公钥
      记录一下密码，后面应该要用到。
    2. 将密钥和github链接
      用户目录下 获取id_rsa.pub文件中内容 复制到github里面 添加ssh
    3. github 创建一个repositories 名称为username.github.io
      (username可自由替换)

    安装hexo

    Code:

    然后找主题啊 放在_themes下 然后修改_config.yml

    (config.yml 里面的theme标签内容换一下就可)

    将本地Hexo博客推送到GithubPages

    1. npm install hexo-deployer-git --save
    2. 修改_config.yml delpoy。repo后的网址就是你刚才创建的repo后面加.git。

    几个常见指令

    1. 更新: hexo g
    2. 发布:hexo d
    3. 本地预览:hexo service
    4. 发布新博客: hexo new post "Hexo 和 github 搭建blog"
      发布的博客会在hexo安装目录下/source/_post/文件夹，自动生成makedown文件。编辑它就好了。
